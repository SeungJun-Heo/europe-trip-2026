# CLAUDE.md

이 파일은 Claude Code(claude.ai/code)가 이 저장소에서 작업할 때 참고할 가이드를 제공합니다.

## 프로젝트 개요

이 프로젝트는 **단일 HTML 파일로 구성된 유럽 가족 여행 가이드북**입니다 (2026년 4월 29일 - 5월 25일, 27일간). 전체 애플리케이션이 `index.html` (826줄)에 담겨 있으며, 빌드 도구, 패키지 매니저, 또는 외부 의존성이 없습니다 (CDN 호스팅 라이브러리 제외).

**여행 루트:** 로마 → 소렌토 → 피렌체 → 베네치아 → 인터라켄 → 파리 → 런던 (7개 도시, 5개 국가)

## 아키텍처

### 파일 구조
- `index.html` - 완전한 독립형 애플리케이션 (HTML + CSS + JavaScript)
- 빌드 프로세스, 프레임워크, 번들러 없음
- 브라우저에서 직접 열기 가능 (`python3 -m http.server` 또는 정적 서버로 제공 가능)

### 기술 스택
- **순수 HTML/CSS/JavaScript** - 프레임워크나 트랜스파일링 없음
- **Leaflet.js** (v1.9.4) - 여행 경로를 표시하는 인터랙티브 지도
- **Google Fonts** - 한글 타이포그래피를 위한 Noto Sans KR
- **외부 이미지** - 도시/관광지 사진을 위한 Pexels.com CDN

### 코드 구성 (index.html 내부)

1. **CSS 스타일** (~9-143줄)
   - 컴포넌트 기반 스타일링: `.header`, `.stats`, `.card`, `.city`, `.hero`, `.accom`, `.tickets`, `.tabs`, `.att-card` 등
   - 색상 체계: 각 도시마다 전용 색상 (로마: #C0392B, 소렌토: #E67E22, 피렌체: #8E44AD 등)
   - 모바일 우선 반응형 디자인 및 인쇄용 미디어 쿼리

2. **HTML 콘텐츠** (145-477줄)
   - 여행 개요 및 도시 네비게이션 링크가 있는 헤더
   - 통계 대시보드 (일수, 도시, 호텔, 뮤지컬)
   - Leaflet 지도 컨테이너 (`#europe-map`)
   - 도시 섹션: 각 도시는 동일한 구조를 따름:
     - 오버레이가 있는 히어로 이미지
     - 예약 번호가 포함된 숙소 카드
     - 티켓 예약 정보 (해당되는 경우)
     - 탭 콘텐츠: 관광지, 일정, 맛집, 포토스팟, 팁
   - 긴급 연락처 정보
   - 여행 체크리스트

3. **JavaScript** (479-826줄)
   - **Leaflet 지도 초기화**: 경로 폴리라인과 도시 마커가 있는 유럽 지도 렌더링
   - **체크리스트 시스템**: `CHECKLIST` 배열이 기본 데이터 정의, `CL` 객체가 런타임 상태 관리
   - **체크리스트 동기화**: GitHub Gist REST API로 크로스 디바이스 동기화 (토큰/Gist ID 하드코딩)
   - **탭 전환**: `switchTab(city, tab, btn, color)` 함수가 도시별 탭 네비게이션 관리

## 콘텐츠 구조

### 도시 데이터 모델
각 도시 섹션은 다음을 포함합니다:
- **히어로 이미지** - 도시 이름, 국가, 날짜, 숙박 일수 배지
- **숙소** 카드 - 이름, 주소, 체크인/체크아웃 시간, 예약 번호, Google Maps 링크
- **티켓** (선택사항) - 날짜/시간이 포함된 사전 예약 관광지
- **탭**:
  - 관광지: 설명과 지도 링크가 있는 이미지 카드
  - 일정: 색상으로 구분된 타임라인이 있는 일일 일정
  - 맛집: 이모지와 설명이 있는 레스토랑 추천
  - 포토스팟: 최고의 사진 촬영 장소
  - 팁: 도시별 여행 팁

### 체크리스트 시스템

체크리스트는 **웹 UI에서 직접 편집 가능**하며, **GitHub Gist로 자동 동기화**됩니다.

- **편집 방법**: 페이지에서 `✏️ 편집` 버튼 → 항목/카테고리 추가·수정·삭제 → `✔ 완료`
- **동기화**: 변경 후 1.2초 뒤 자동으로 GitHub Gist에 저장, 다른 기기에서 열면 자동 반영
- **저장 위치**: GitHub Gist (`b755796c96803e2a453615dd89394f05`) + localStorage 캐시 (오프라인 대비)
- **기본 데이터**: `CHECKLIST` 배열이 Gist가 비어있을 때의 폴백 데이터 역할

코드로 기본 항목을 수정하려면 `CHECKLIST` 배열을 편집한 뒤 Gist도 함께 업데이트해야 합니다.

## 중요 컨텍스트

- **언어**: 콘텐츠는 한국어 (한국 여행자를 위한 가족 가이드북)

- **날짜 형식**: 한국 스타일 (4/29, 5/25), 24시간 형식
